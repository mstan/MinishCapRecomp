# generated/

**Do not hand-edit anything in this directory.**

Files here are produced by `gba_recompile` from
[`../gbarecomp`](../../gbarecomp). If output looks wrong:

1. Fix the recompiler, the runtime, or `game.toml`.
2. Re-run `gba_recompile`.
3. Rebuild.

Editing generated C to "fix" a symptom is the fastest way to produce
half-finished edges that future work trips over. See
`../../gbarecomp/CLAUDE.md`.
